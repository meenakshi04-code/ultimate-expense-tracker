# 💰 Ultimate Expense Tracker

An advanced and visually interactive **Expense Tracker Web App** built using **HTML, CSS, and JavaScript**.

This project goes beyond a basic tracker by combining **UI animations, insights, local storage, and smart feedback system**.

---

## 🚀 Features

### 💸 Expense Management

* Add and delete expenses easily
* Category-based tracking
* Real-time total calculation

### 🎨 Dynamic UI Experience

* Background changes based on category
* Live image preview
* Glassmorphism design
* Smooth animations & transitions

### 🧠 Smart Insights

* 📊 Detects top spending category
* 💡 Gives intelligent spending insights
* ⚠ Alerts when overspending

### 🐰 Interactive Assistant

* Floating bunny guide 🐰
* Smart messages based on spending
* Popup warning for high expenses

### 💾 Data Persistence

* Saves data using **LocalStorage**
* Data remains after page refresh

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations, Glass UI, Gradients)
* JavaScript (DOM, LocalStorage, Logic)
